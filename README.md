# TransitHubJU

A smart campus transportation management system designed for Jahangirnagar University. TransitHubJU aims to simplify ride sharing and campus transportation by connecting passengers and drivers through a centralized platform.

> 🚧 This project is currently under active development.

## Project Status

We are currently working on:

- 📄 Software Requirements Specification (SRS)
- 📚 Project Documentation
- 📝 Coding Standards
- 📖 GitHub Wiki
- 🎨 UI/UX Design
- 🏗️ System Architecture
- 💻 Application Development

## Planned Features

- Passenger ride requests
- Driver ride management
- Passenger queue system
- Ride notifications
- Seat availability tracking
- User authentication
- Trip history
